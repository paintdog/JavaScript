addEventListener()

Mit dem window.addEventListener() kann ich auf ein Event in einem Fenster reagieren und das Event an eine Funktion oder ein Objekt weiterreichen (weitere Beispiele sind button.addEventListener()).

Ausgabe

Ausgaben mittels JavaScript können mittels console.log(), document.write() vorgenommen werden und, indem man einem HTML-Element der Webseite einen neuen Wert (...innerHTML = "") zuweist.

Einbindung von JavaScript

Ich kann JavaScript in script-Tags am Ende einer HTML-Seite innerhalb des body-Tags einbinden oder als externe Datei (Beispiel).

Elemente erzeugen

Mit dem Befehl document.createElement(...) kann ich ein HTML-Element in einer Seite erzeugen.

Formatierungen

Mit setAttribute() kann ich style-Eigenschaften ändern.

Selektoren

Mittels document.getElementById() kann ich ein Element anhand seiner ID auswählen und mittels document.querySelector() kann ich das erste Element auf einer Seite von einem bestimmten Element-Typ wählen. Mittels innerHTML kann ich den Text zwischen den HTML-tags erhalten oder ändern.
